# WeChat_AES_Encrypt

微信AES端对端自动加密脚本，实现自动加解密聊天

**原理**

使用AES-256位加密微信聊天消息，通过sha256（或者RSA）生成AES密钥!

先在客户端加密后传送。

多线程操作，自动加解密！

**使用**
安装好相应的库

可以自行修改源码设计密钥生成方式（例如利用RSA生成密钥）

扫码登陆，输入需要加密聊天的对象的昵称即可开始自动加密聊天模式
